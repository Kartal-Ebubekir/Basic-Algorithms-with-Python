# Python ile Veri Yapıları ve Algoritmalar 

Bu proje, Yazılım Mühendisliği ve Bilgisayar Bilimlerinin temeli olan *Veri Yapıları ve Algoritmalar* konularını Python programlama dili kullanarak ele alan kapsamlı bir rehberdir.

Proje, hem teorik zaman karmaşıklığı analizlerini (Big O Notation) hem de popüler arama ve sıralama algoritmalarının adım adım kod implementasyonlarını içerir. Proje içerisinde algoritmaların çalışma mantığını daha iyi kavramak için animasyonlar ve şemalar kullanılmıştır. Kod blokları, her satırın ne işe yaradığını anlatan detaylı yorum satırları içerir.

## İçerik

Bu çalışma not defteri (notebook) aşağıdaki ana başlıkları kapsar:

### 1. Algoritma Analizi ve Big O Notasyonu
* **Zaman Karmaşıklığı (Time Complexity - T(n)):** Algoritmaların performans analizi.
* **Asimptotik Notasyonlar:** Big O (O), Big Omega (Ω) ve Big Theta (ϴ).
* **Uzaysal Karmaşıklık (Space Complexity - S(n)):** Bellek yönetimi analizi.

### 2. Arama Algoritmaları (Searching)
* **Linear Search:** O(n) karmaşıklık.
* **Binary Search:** O(log n) karmaşıklık (Sıralı dizilerde).

### 3. Sıralama Algoritmaları (Sorting)
Her algoritmanın çalışma mantığı, Python kodu ve görselleştirmeleri ile anlatılmıştır:

* **Bubble Sort:** Kabarcık sıralaması.
* **Selection Sort:** Seçmeli sıralama.
* **Insertion Sort:** Araya eklemeli sıralama.
* **Quick Sort:** Böl ve yönet (Divide and Conquer) mantığı.
* **Merge Sort:** Birleştirmeli sıralama.
* **Shell Sort:** Insertion sort'un optimize edilmiş hali.
* **Radix Sort:** Basamak tabanlı sıralama (Integer sorting).
* **Heap Sort:** Yığın (Heap) veri yapısı kullanarak sıralama.

# Data Structures and Algorithms with Python (DSA)

This project is a comprehensive guide covering the fundamentals of Software Engineering and Computer Science: *Data Structures & Algorithms*, using the Python programming language.

The project includes both theoretical time complexity analyses (Big O Notation) and step-by-step code implementations of popular searching and sorting algorithms. Animations and schemas are used within the project to better understand the logic of the algorithms. Code blocks contain detailed comments explaining what each line does.

## Content

This notebook covers the following main topics:

### 1. Algorithm Analysis and Big O Notation
* **Time Complexity (T(n)):** Performance analysis of algorithms.
* **Asymptotic Notations:** Big O (O), Big Omega (Ω), and Big Theta (ϴ).
* **Space Complexity (S(n)):** Memory management analysis.

### 2. Searching Algorithms
* **Linear Search:** O(n) complexity.
* **Binary Search:** O(log n) complexity (on sorted arrays).

### 3. Sorting Algorithms
The logic, Python code, and visualizations for each algorithm are explained:

* **Bubble Sort**
* **Selection Sort**
* **Insertion Sort**
* **Quick Sort:** Divide and Conquer logic.
* **Merge Sort**
* **Shell Sort:** Optimized version of insertion sort.
* **Radix Sort:** Digit-based sorting.
* **Heap Sort:** Sorting using the Heap data structure.
